### Examples

Basic button:

```js
<Button>Push Me</Button>
```

Secondary button:

```js
<Button
  variant='secondary'
>Push Me</Button>
```

Disabled button:

```js
<Button
  disabled={true}
>Push Me</Button>
```

Block button:

```js
<Button
  block={true}
>Push Me</Button>
```

Small button:

```js
<Button
  size='sm'
>Push Me</Button>
```

Large button:

```js
<Button
  size='lg'
>Push Me</Button>
```
